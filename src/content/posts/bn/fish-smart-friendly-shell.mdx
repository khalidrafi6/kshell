fish shell: একটি সহজ ও চমৎকার শেল

fish (Friendly Interactive Shell) হচ্ছে bash বা zsh এর মতই একটি কমান্ড লাইন শেল। শেল হচ্ছে একটা কমান্ড লাইন ইন্টারপ্রেটার যা আমাদেরকে বিভিন্ন কমান্ড দিয়ে কম্পিউটার চালাতে সহায়তা করে। যদি শেল সম্পর্কে আপনার খুব একটা জানাশোনা না থাকে, তাহলে খুব সম্ভবত আপনি bash ব্যবহার করছেন কারণ লিনাক্স ডিস্ট্রোগুলোতে সাধারণত এটাই ইন্সটল করে দেয়া থাকে। fish একটা ইউজার-বান্ধব শেল। এটার দারুন কিছু ফিচার আছে যা কোন কনফিগারেশন ছাড়াই কাজ করে। যদি আপনি কোন সিনট্যাক্স বা কনফিগারেশন না শিখেই নিজের কমান্ড লাইনকে আরেকটু স্মার্ট ও সুন্দর বানাতে চান, তাহলে fish ব্যবহার করতে পারেন। চলুন একনজরে এটার কিছু চমৎকার সুবিধা দেখে নেয়া যাক:

### Autosuggestions

fish আপনার পূর্ববর্তী কমান্ড ও কমপ্লিশনের ওপর ভিত্তি করে আপনাকে কমান্ড লেখার সময় সাজেশন দিবে। এই সুবিধার কারণে আপনাকে এক কমান্ড বারবার লিখতে হবে না। এটা ফাইল লোকেশন, এমনকি কমান্ড অপশনও কমপ্লিট করতে পারে।

### Tab Completion:

fish উন্নত মানের Tab Completion সাপোর্ট করে। bash এ sudo লেখার পর আর Tab Completion কাজ করে না। কিন্তু fish এ সেই সমস্যা নেই। এখানে কমান্ডগুলোর একটা সংক্ষিপ্ত বিবরণও দেয়া থাকে। আপনি দ্বিতীয়বার Tab চেপে অপশন বাছাই করতে পারবেন। git কমান্ডসহ অনেক কমান্ডই fish কম্পলিট করতে পারে।

### Syntax Highlighting:

কোন কনফিগারেশন ছাড়াই fish শেলে Syntax Highlighting কাজ করে। আপনি কমান্ড লেখার সময়ই fish সেটা হাইলাইট করতে থাকবে। ভুল কমান্ডকে লাল রঙে চিহ্নিত করা হয়। আর সঠিক কমান্ডকে অন্য একটি রঙে। পাশাপাশি, সঠিক ফাইল লোকেশন লিখলে তার নিচে দাগ দেয়া হয়। এ দ্বারা বোঝা যায় ফাইল/ফোল্ডারটি সিস্টেমে আছে।

### Universal Variables

Universal Variable হচ্ছে এমন এক ধরনের variable যেটা fish শেলের সব সেশনেই থাকে এবং রিবুট করলেও এর মান বদলায় না। fish অনেক কনফিগারেশন অপশন Universal Variable হিসেবে রাখে। যার ফলে, একটা কনফিগারেশনের variable একবার পরিবর্তন করলেই সেই সেটিংটা তৎক্ষণাৎ এবং স্থায়ীভাবে বদলে যাবে। Universal Variable সরাসরি দেখার জন্য দুইটা fish উইন্ডো খুলুন। তারপর, যেকোন একটাতে নিচের কমান্ড রান করুন:

set fish_color_cwd blue

যেহেতু, fish_color_cwd একটা Universal Variable, তাই বর্তমান working directory এর রঙ দুইটা উইন্ডোতেই বদলে যাবে।

একই কারণে fish শেলে $PATH চলকে কোন ডিরেক্টরি যুক্ত করাও খুবই সহজ। যেমন: /usr/local/bin কে $PATH চলকে স্থায়ীভাবে যুক্ত করতে লিখুন:

fish_add_path /usr/local/bin

এটার সুবিধা হচ্ছে একটা সামান্য কাজে আপনার ফাইল ঘাঁটাঘাঁটি করা লাগছে না। এটা $fish_user_paths নামক Universal Variable ব্যবহার করে, যেটার মান $PATH চলকের শুরুতে আপনা-আপনি যুক্ত করা হয়। যার কারণে সিস্টেম রিবুট করার পরেও $PATH এর নতুন মান থাকবে।

### Web-based Configuration

fish শেল Web-based Configuration সাপোর্ট করে। fish_config কমান্ড রান করলে আপনার ব্রাউজারে এই কনফিগারেশন ইন্টারফেসটি খুলবে। এখান থেকে সহজেই শেলের colors, prompt বদলানো যায়, এবং variable, function ইত্যাদি দেখা যায়।

তাছাড়াও এই শেলে abbreviations, Vi mode ইত্যাদি সুবিধাও আছে। fish সহজে সেটাপ করতে, আপনার টার্মিনালে প্রধান শেলের পরিবর্তে সরাসরি fish চালু করার সেটিং অন করে দিন। তার জন্য আপনার সিস্টেমের কোথায় এটি ইন্সটল করা হয়েছে তা খুঁজে বের করতে হবে। এই শেল সম্পর্কে বিস্তারিত জানতে তাদের ওয়েবসাইট (http://fishshell.com/) ভিজিট করুন।
