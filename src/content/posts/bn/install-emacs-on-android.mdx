অ্যান্ড্রয়েডে কোডিং করার জন্য [Emacs](https://www.gnu.org/software/emacs/) একটি চমৎকার এডিটর। এটি ক্রস-প্ল্যাটফর্ম, তাই আপনি সব ডিভাইসে একই কোড এডিটর ও পরিচিত পরিবেশে কোডিং করার সুবিধা পাবেন। আপনার যদি [কম্পিউটার ছাড়াই কোডিং](/bn/coding-without-computer) করার প্রয়োজন পড়ে, তাহলে এটিই সেরা। আমি অ্যান্ড্রয়েডে আর কোনো কোড এডিটর চিনি না যা ডেস্কটপেও চালানো যায় এবং Emacs এর মতো শক্তিশালী। Emacs এর প্রকাশক এডিটরটির পরিচয়ে এই বৈশিষ্ট্যগুলোর কথা বলে থাকেন:

- **Advanced:** Emacs সাধারণ লেখালেখির বাইরেও ইনডেন্ট করা, একাধিক ফাইল এডিট করা, রিমোট ফাইল এডিট করা ইত্যাদি কাজ পারে।

- **Self-documenting:** এডিটরটির মধ্য থেকেই আপনি হেল্প কমান্ড দিয়ে কোন অপশনের কাজ কি বা একটি টপিকের সাথে সম্পৃক্ত সকল কমান্ড খুঁজে পেতে পারেন।

- **Customizable:** সহজেই এর কমান্ডগুলো বদলাতে পারবেন।

- **Extensible:** কেবল কমান্ড বদলানো ছাড়াও নতুন কমান্ড বানাতে পারবেন।

আপনি [Termux](https://termux.dev/) এর মাধ্যমে Emacs এ `git`, `python` ইত্যাদি প্যাকেজ ব্যবহার করতে পারবেন। আর, আসলে Emacs এ আপনি শুধু কোড এডিটিং ছাড়াও আরো অনেক কিছুই করতে পারবেন। ওয়েব ব্রাউজিং, ইমেইল পড়া, হিসাব-নিকাশ, কর্মপরিকল্পনাসহ আরো অনেক কিছুই করতে পারবেন। হতে পারে Emacs ব্যবহার শুরু করলে আপনার ফোন থেকে অনেক অ্যাপ আনইনস্টল করে দিতে ইচ্ছে করবে! [Emacs Lisp](https://www.gnu.org/software/emacs/manual/html_node/eintr/index.html) দিয়ে প্রোগ্রাম করে বা [প্যাকেজ](https://elpa.gnu.org/) ইন্সটল করে Emacs এ আপনি আপনার মনমত অনেক কিছুই করতে পারবেন যা সাধারণ এডিটর পারে না।

চলুন, এবার অ্যান্ড্রয়েডে Emacs ইনস্টল করার প্রক্রিয়ায় যাওয়া যাক। অ্যান্ড্রয়েডে Emacs ইনস্টল করার তিনটি পদ্ধতি আছে। তবে আমি এই টিউটোরিয়ালে শুধু দুটি নিয়ে আলোচনা করব। আর, তার মধ্যে [Termux সাপোর্ট সহ Emacs ইনস্টল](#termux-সাপোর্ট-সহ-ইনস্টল) করাই সেরা পদ্ধতি।

## ইনস্টল প্রক্রিয়া

### সাধারণ ইনস্টল

এটি সবচেয়ে সহজ এবং সোজা পদ্ধতি। আপনি শুধু অফিসিয়াল সোর্স থেকে অ্যাপটি ডাউনলোড করে ইনস্টল করবেন। ব্যস! তবে এতে `git`, `ssh`, `python` ইত্যাদি অতিরিক্ত প্যাকেজ পাবেন না। সাধারণ কোড এডিটিং এর জন্য আপনি এই পদ্ধতি ব্যবহার করতে পারেন। প্রক্রিয়াটি হল:

1. [আপনার নিকটতম GNU মিরর](http://ftpmirror.gnu.org/gnu/emacs/android) বা [মূল GNU FTP সার্ভার](http://ftp.gnu.org/gnu/emacs/android) ভিজিট করুন।

2. আপনার ফোনের CPU আর্কিটেকচার এবং অ্যান্ড্রয়েড ভার্সন অনুযায়ী একটি প্যাকেজ ডাউনলোড করুন।

3. এখন, `.apk` ফাইলটি ইনস্টল করুন। Unknown Source থেকে ইনস্টল করার অনুমতি চাইলে দিয়ে দিন।

**নোট:** [F-Droid-এও](https://f-droid.org/) একটি ভার্সন পাওয়া যায়। তবে এটি অফিসিয়াল ভার্সন থেকে পুরানো। আর, এতে অনেক বাগ ফিক্স ও নতুন ফিচার নেই। তাই অফিসিয়াল APK ব্যবহার করাই ভালো।

### Termux সাপোর্ট সহ ইনস্টল

আপনি Termux সাপোর্ট সহও Emacs ইনস্টল করতে পারেন। তাহলে Emacs থেকে Termux প্যাকেজ যেমন `git`, `clang`, `python`, `fish` ইত্যাদি চালানো যাবে। আমি এই পদ্ধতিটাই সেরা মনে করি। কারণ এটি Emacs কে Termux এর সাথে যুক্ত করে আপনার কোডিং অভিজ্ঞতা উন্নত করে। তার জন্য আপনার Emacs-এর কী দিয়ে সাইন করা Termux প্যাকেজ দরকার। Emacs ডেভেলপাররাই এমন প্যাকেজ প্রদান করেন। ইন্সটল করার প্রক্রিয়াটি নিম্নরূপ:

1. আপনার ফোনে আগের Termux এবং Emacs অ্যাপ থাকলে সরিয়ে ফেলুন।

2. প্রথমে [এই সোর্স](https://sourceforge.net/projects/android-ports-for-gnu-emacs/files/termux/termux-app_apt-android-7-release_universal.apk/download) থেকে Termux ইনস্টল করুন।

3. তারপর [আপনার নিকটতম মিরর](http://ftpmirror.gnu.org/gnu/emacs/android) বা [FTP সার্ভার](http://ftp.gnu.org/gnu/emacs/android) থেকে Emacs ইনস্টল করুন। আপনাকে অবশ্যই `termux/` ডিরেক্টরি থেকে প্যাকেজ ইনস্টল করতে হবে।

4. এখন, Termux এ নিচের কমান্ড রান করুন:

```shell
$ pkg update && pkg upgrade
```
এখানে প্রতিটি প্রশ্নের যথাযথ উত্তর দিন।

5. সব প্যাকেজ ইনস্টল এবং আপডেট হওয়ার পর, Emacs-এ গিয়ে নিচের কোডটি `.emacs.d/early-init.el` ফাইলে যোগ করুন (Early Init File সম্পর্কে আরো জানতে [Emacs ম্যানুয়াল](https://www.gnu.org/software/emacs/manual/html_node/emacs/Early-Init-File.html) দেখুন):

```elisp
(setenv "PATH" (format "%s:%s" "/data/data/com.termux/files/usr/bin"
                        (getenv "PATH")))
(push "/data/data/com.termux/files/usr/bin" exec-path)
```
এটি Emacs এর ভেতরে Termux বাইনারিগুলোর লোকেশন যোগ করবে।

এবার, Emacs রিস্টার্ট করুন। আপনি এখন Emacs থেকে Termux এর প্রোগ্রাম ব্যবহার করতে পারবেন।

**নোট:** Emacs ও Termux (বা এর প্লাগ-ইন) এর যেকোনো ভবিষ্যত আপডেট একই কী দিয়ে সাইন করা থাকতে হবে। GNU সার্ভার থেকে ডাউনলোড করলে আলাদা করে সাইন করার প্রয়োজন নেই। তবে অন্য কোথাও থেকে প্যাকেজ ডাউনলোড করলে আপনাকে অবশ্যই নিচের লিংকের কী দিয়ে সাইন করে নিতে হবে। এটা Emacs সার্ভারে ব্যবহার করা হয়:

https://git.savannah.gnu.org/cgit/emacs.git/tree/java/emacs.keystore

### Termux রিপোজিটরি থেকে ইনস্টল

আপনি সরাসরি Termux রিপোজিটরি থেকেও Emacs ইনস্টল করতে পারেন। তারপর Termux এর ভিতর থেকে Emacs ব্যবহার করতে পারবেন। তবে, আমার মতে অফিসিয়াল অ্যাপই উত্তম। কারণ, এটি অ্যান্ড্রয়েড সিস্টেমের মধ্যেই একটি গ্রাফিক্যাল অ্যাপ হিসেবে চলে। এটাতে অ্যান্ড্রয়েডের জন্য ভালো সাপোর্ট আছে। আমি Termux-এর Emacs প্যাকেজ ব্যবহার করি না এবং এখানে এই পদ্ধতি আলোচিত হবে না। আপনি অনলাইনে সার্চ করে টিউটোরিয়াল খুঁজে পেতে পারেন।

## প্রাথমিক জ্ঞান ও সেটআপ

### অ্যান্ড্রয়েডে Emacs কাস্টমাইজ করা

অ্যান্ড্রয়েডে Emacs কাস্টমাইজ করার সেরা উপায় হলো [Easy Customization Interface](https://www.gnu.org/software/emacs/manual/html_node/emacs/Easy-Customization.html) ব্যবহার করা। মেনু বারের **Options > Customize Emacs** থেকে এটিতে যাওয়া যায়। **Customize Emacs** মেনুতে অনেক অপশন আছে। আপনি **Top-level Emacs Customization Group** দিয়ে শুরু করতে পারেন, যা মূলত **Easy Customization Interface** এর প্রথম পাতা। সেটিংস সার্চ করার জন্য এর উপরে একটি সার্চ বারও আছে। তাছাড়া, আপনি **All Settings Matching** অপশন ব্যবহার করেও সার্চ করতে পারেন।

Emacs কে [initialization file](https://www.gnu.org/software/emacs/manual/html_node/emacs/Init-File.html) দিয়েও কাস্টমাইজ করা যায়। `~/.emacs.el`, `~/.emacs`, বা `~/.emacs.d/init.el` - এই নামগুলো ব্যবহার করে Emacs তার init ফাইল খোঁজে। আপনি এই ফাইলে Lisp প্রোগ্রামিং ভাষায় লিখে Emacs সেটিংস নিয়ন্ত্রণ করতে পারেন। উদাহরণস্বরূপ, টাইপিং বন্ধ করার ৫ সেকেন্ড পর অটো-সেভ করতে `auto-save-timeout` চলকের মান `setq` দিয়ে পরিবর্তন করুন:

```elisp
(setq auto-save-timeout 5)
```

মডিফায়ার বার চালু করতে `modifier-bar-mode` নামক মাইনর মোড কমান্ডকে আর্গুমেন্ট `1` দিয়ে কল করুন:

```elisp
(modifier-bar-mode 1)
```

আরো বিস্তারিত জানতে [Emacs ম্যানুয়াল](https://www.gnu.org/software/emacs/manual/html_node/emacs/Init-Syntax.html) দেখুন।

তাছাড়া Emacs-এর ভেতরে আপনি যেকোন সময় ‍`M-x` চেপে যেকোন কমান্ড খুঁজে বের করতে পারেন।

### Emacs কে স্টোরেজ ব্যবহারের অনুমতি দিন

Emacs এমনিতেই তার অ্যাপ ডাটা ডিরেক্টরি (সাধারণত `/data/data/org.gnu.emacs/files`) ব্যবহার করতে পারে যা তার হোম ডিরেক্টরি হিসেবে ব্যবহৃত হয়। তবে, ফাইল খোলা এবং এডিট করার জন্য আপনার অভ্যন্তরীণ স্টোরেজ (`/sdcard`) ব্যবহারের অনুমতি প্রয়োজন। অনুমতি দিতে অ্যান্ড্রয়েড ১১+ এর ক্ষেত্রে:

1. **Settings** থেকে **Apps** সেকশনে যান।

2. তারপর, **Special App Access** অপশনে যান।

3. অ্যাপের তালিকায় Emacs খুঁজে তার জন্য **All files access** চালু করুন।

অ্যান্ড্রয়েড ১১ এর নিচের ভার্সনের জন্য, আপনি **App Info** এর **Permissions** মেনু থেকেই স্টোরেজ ব্যবহারের অনুমতি দিতে পারবেনন।

আপনি Emacs কে অন্যান্য অ্যাপের স্টোরেজ (যেমন ক্লাউড স্টোরেজ অ্যাপ) ব্যবহারের অনুমতিও দিতে পারেন। তার জন্য Emacs মেনু বার থেকে **File > Execute command** এ গিয়ে `android-request-directory-access` কমান্ড দিন। এখন, সংশ্লিষ্ট স্টোরেজ এবং তারপর আপনার পছন্দের ডিরেক্টরি বেছে নিন।

### সর্বদা অন-স্ক্রীন কিবোর্ড চালু রাখুন

অ্যান্ড্রয়েডে Emacs রিড-অনলি বাফারে অন-স্ক্রীন কিবোর্ড লুকিয়ে রাখে যাতে স্ক্রীনের জায়গা বাঁচানো যায়। আপনি কিবোর্ড দিয়ে যা করতে চান তা সাধারণত মেনু বার বা টুল বার থেকেই করে নিতে পারবেন । তবুও, কিবোর্ড সর্বদা হাতের কাছে থাকা খুবই উপকারী। তার জন্য `Customize` এ গিয়ে `touch-screen-display-keyboard` চালু করুন।

### কিবাইন্ডিং নির্ভর কিছু প্যাকেজের সাপোর্টের জন্য

কিছু Emacs প্যাকেজ (যেমন evil) যেগুলো কিবাইন্ডিংয়ের উপর নির্ভর করেে, সেগুলো অ্যান্ড্রয়েডের অন-স্ক্রীন কিবোর্ডের সাথে কাজ করে না। কারণ অ্যান্ড্রয়েড ইনপুট মেথড Emacs এর টেক্সট কনভার্সন স্টাইলের উপর নির্ভর করে যা Emacs কে কিছু ফিচার যুক্ত করতে সুবিধা দেয়। আপনি যদি এমন কোনো প্যাকেজ ব্যবহার করেন যা এ কারণে কাজ করে না, তাহলে `overriding-text-conversion-style` চলক দিয়ে গ্লোবালি টেক্সট কনভার্সন বন্ধ করুন অথবা `text-conversion-style` দিয়ে লোকালি নির্দিষ্ট বাফারে বন্ধ করুন।

### দৃশ্যমান বারগুলো সেট আপ করুন

আপনি যদি ফিজিক্যাল কিবোর্ড ছাড়া অ্যান্ড্রয়েডে Emacs ব্যবহার করতে চান, বিভিন্ন দৃশ্যমান বার যেমন মেনু বার এবং টুল বার হবে আপনার সেরা বন্ধু। তাই `Customize` থেকে দুটোই চালু করুন। অ্যান্ড্রয়েডের জন্য একটি খুবই উপকারী অপশন হলো Emacs এর টুল বার এর সাথের মডিফায়ার বার। এতে মডিফায়ার কী (যেমন CTRL, ALT, META ইত্যাদি) এর বাটন থাকে। আপনি `tool-bar-mode` চালু থাকা অবস্থায় `modifier-bar-mode` দিয়ে এটি চালু করতে পারেন। যখনই কোনো মডিফায়ার কী টাচ করবেন, Emacs অন-স্ক্রীন কিবোর্ডও দেখাবে এবং টেক্সট কনভার্সন সাময়িকভাবে বন্ধ করে দেবে, যাতে আপনি Text বা Prog Mode বাফারে সেগুলো ব্যবহার করতে পারেন। আপনি `tool-bar-position` এর মান `bottom` সেট করে এই কী-গুলো আপনার আঙুলের কাছাকাছি রাখতে পারেন।

### Emacs দিয়ে ফাইল ওপেন করা

আপনি যখন অ্যান্ড্রয়েডে কোন ফাইল ওপেন করার চেষ্টা করেন, এটি জিজ্ঞাসা করে কোন অ্যাপ ব্যবহার করতে হবে। Emacs দিয়ে কোন ফাইল ওপেন করতে আপনার Emacs সার্ভার চালু করতে হবে। Emacs এর ভিতর থেকে `M-x server start` রান করে সার্ভার চালু করতে হয়। তবে Emacs খুললেই সার্ভার অটোমেটিকালি চালু করা আপনার জন্য সুবিধাজনক হবে। এটি করতে, আপনার [initialization file](https://www.gnu.org/software/emacs/manual/html_node/emacs/Init-File.html) এ `(server-start)` যোগ করুন।

### পড়া ও এডিটিং উন্নত করা

আমরা অনেক সময় লম্বা লাইনগুলো ওয়ার্ড-র‍্যাপ দিয়ে কয়েক লাইনে দেখি যাতে ডানে-বামে স্ক্রল করতে না হয়। `visual-line-mode` দিয়ে আপনি লেখার লাইনগুলো প্রকৃত ক্রমের বদলে দৃশ্যমান ক্রমানুসারে এডিট করতে পারবেন। এই মোড সাধারণ এডিটিং কমান্ডগুলোকে দৃশ্যমান লাইনে কাজ করতে দেয় এবং বাফারে ওয়ার্ড-র‍্যাপও চালু করে। ফলে আপনি অ্যান্ড্রয়েডে ভালোভাবে পড়তে এবং টাইপ করতে পারবেন। আপনি `visual-wrap-prefix-mode` চালু করে পরবর্তী লাইনগুলো ইন্ডেন্টও করতে পারেন। পাশাপাশি, আপনি নিচের ফিচারগুলোও দেখতে পারেন:

- [Visual fill](https://elpa.gnu.org/packages/visual-fill.html)

- [Adaptive filling](https://www.gnu.org/software/emacs/manual/html_node/emacs/Adaptive-Fill.html)

- [Adaptive wrap](https://elpa.gnu.org/packages/adaptive-wrap.html)

### ফন্ট সেট করা

আপনি `set-frame-font` ফাংশন দিয়ে Emacs এ ডিফল্ট ফন্ট সেট করতে পারেন:

```elisp
(set-frame-font "JetBrains Mono")
```

পাশাপাশি, আপনি `Customize` এর **Faces** গ্রুপ থেকেও ফন্ট সেট করতে পারবেন। Emacs এর বিভিন্ন অংশে ব্যবহৃত ফন্টের জন্য এই গ্রুপের অধীনে বিভিন্ন সাবগ্রুপ আছে। আপনি `Basic Faces` সাবগ্রুপ দিয়ে শুরু করতে পারেন এবং প্রয়োজন অনুযায়ী অন্যগুলো কাস্টমাইজ করতে পারেন।

### Emacs ম্যানুয়াল পড়ুন

Emacs এ অনেক ফিচার রয়েছে এবং আপনার মাঝে মাঝেই সাহায্যের প্রয়োজন হবে। আর সাহায্য পাওয়ার সেরা উপায় হলো ম্যানুয়াল দিয়ে "নিজের সাহায্য নিজে করা"। Emacs এর সাথেই বেশ কিছু ম্যানুয়াল অন্তর্ভুক্ত আছে:

- [The Emacs Manual](https://www.gnu.org/software/emacs/manual/html_node/emacs/index.html)

- [An Introduction to Programming in Emacs Lisp](https://www.gnu.org/software/emacs/manual/html_node/eintr/index.html)

- [Emacs Lisp Reference Manual](https://www.gnu.org/software/emacs/manual/html_node/elisp/index.html)

- এবং [অন্যান্য ম্যানুয়াল](https://www.gnu.org/software/emacs/manual/index.html)

আপনি `C-h i` কমান্ড দিয়ে Emacs এর ভিতর থেকে বা [অনলাইনে](https://www.gnu.org/software/emacs/manual/index.html) এগুলো পড়তে পারেন। `C-h r` কমান্ড Emacs ম্যানুয়াল ওপেন করে। ম্যানুয়ালের ভিতর `i` টাইপ করে আপনি হেল্প টপিক খুঁজে বের করতে পারেন যা খুবই সুবিধাজনক। তাছাড়া আপনি `describe-*` কমান্ড যেমন `describe-function`, `describe-mode`, `describe-variable` ইত্যাদি দিয়ে বিভিন্ন ফাংশন, চলক, মোড ইত্যাদির বিবরণ এবং তাদের মান সম্পর্কে জানতে পারেন। শুরু করতে শুধু `M-x` টাইপ করুন বা **File > Execute Command** অপশন ব্যবহার করুন।

আপনি কি প্রোগ্রামিং শিখতে চান কিন্তু আপনার কাছে কম্পিউটার নেই? আমি আপনাকে অ্যান্ড্রয়েড ফোনে Emacs দিয়ে একটি কোডিং এনভায়রনমেন্ট সেট আপ করতে সাহায্য করতে পারি একটি ছোট ফির বিনিময়ে। আমি [IT সার্ভিস](services) ও প্রদান করি যা IT সাপোর্ট থেকে ওয়েব ডেভেলপমেন্ট পর্যন্ত বিস্তৃত। যদি প্রযুক্তি ব্যবহার সংক্রান্ত সমস্যায় সাহায্য দরকার হয়, যদি আপনি এটি শিখতে চান বা যদি আপনার নিজের ওয়েবসাইট ডেভেলপ করতে চান, আমাকে [এখানে](mailto:khalids_shell.l3rmw@slmail.me) কন্টাক্ট করুন।
