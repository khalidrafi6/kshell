অ্যান্ড্রয়েডে কোডিং করার জন্য [Emacs](https://www.gnu.org/software/emacs/) একটি চমৎকার এডিটর। এটি ক্রস-প্ল্যাটফর্ম, তাই আপনি সব ডিভাইসে একই কোড এডিটর ও পরিচিত পরিবেশে কোডিং করার সুবিধা পাবেন। আপনার যদি [কম্পিউটার ছাড়াই কোডিং](/bn/coding-without-computer) করার প্রয়োজন পড়ে, তাহলে এটিই সেরা এডিটর। যদিও এটি তুলনামূলক নতুন। গত ফেব্রুয়ারি মাসে Emacs এর অ্যান্ড্রয়েড ভার্সন প্রকাশিত হয়। তবে আমি অ্যান্ড্রয়েডে আর কোনো কোড এডিটর চিনি না যা ডেস্কটপেও চালানো যায় এবং Emacs এর মতো শক্তিশালী। Emacs এর প্রকাশক এডিটরটির পরিচয় দিয়েছেন এভাবে:

> Emacs is the advanced, extensible, customizable, self-documenting editor

অর্থাৎ, এর মূল বৈশিষ্ট্যগুলো হচ্ছে:

- **Advanced:** Emacs সাধারণ লেখালেখির বাইরেও ইনডেন্ট করা, একাধিক ফাইল এডিট করা, রিমোট ফাইল এডিট করা ইত্যাদি কাজ পারে।

- **Self-documenting:** এডিটরের মধ্য থেকেই আপনি হেল্প কমান্ড দিয়ে কোন অপশনের কাজ কি বা একটি টপিকের সাথে সম্পৃক্ত সকল কমান্ড খুঁজে পেতে পারেন।

- **Customizable:** সহজেই এর কমান্ডগুলো বদলাতে পারবেন।

- **Extensible:** কেবল কমান্ড বদলানো ছাড়াও নতুন কমান্ড বানাতে পারবেন।

<br />
আপনি [Termux](https://termux.dev/) এর মাধ্যমে Emacs এ `git`, `python` ইত্যাদি প্যাকেজ ব্যবহার করতে পারবেন। আর, আসলে Emacs এ আপনি শুধু কোড এডিটিং ছাড়াও আরো অনেক কিছুই করতে পারবেন। ওয়েব ব্রাউজিং, ইমেইল্ বা সংবাদ পড়া, হিসাব-নিকাশ, কর্ম-পরিকল্পনাসহ আরো অনেক কিছুই করতে পারবেন। হতে পারে Emacs ব্যবহার শুরু করলে আপনার ফোন থেকে অনেক অ্যাপ আনইনস্টল করে দিতে ইচ্ছে করবে! [Emacs Lisp](https://www.gnu.org/software/emacs/manual/html_node/eintr/index.html) দিয়ে প্রোগ্রাম করে বা [প্যাকেজ](https://elpa.gnu.org/) ইন্সটল করে আপনি এমন অনেক কিছুই করতে পারবেন যা সাধারণ এডিটর পারে না।

চলুন, এবার অ্যান্ড্রয়েডে Emacs ইনস্টল করার প্রক্রিয়ায় যাওয়া যাক। অ্যান্ড্রয়েডে Emacs ইনস্টল করার তিনটি পদ্ধতি আছে। তবে আমি এই টিউটোরিয়ালে শুধু দুটি নিয়ে আলোচনা করব। আর, তার মধ্যে [Termux সাপোর্ট সহ Emacs ইনস্টল](#termux-সাপোর্ট-সহ-ইনস্টল) করাই সেরা পদ্ধতি।

## ইনস্টল প্রক্রিয়া

### সাধারণ ইনস্টল

এটি সবচেয়ে সহজ এবং সোজা পদ্ধতি। আপনি শুধু অফিসিয়াল সোর্স থেকে অ্যাপটি ডাউনলোড করে ইনস্টল করবেন। ব্যস! তবে এতে `git`, `ssh`, `python` ইত্যাদি অতিরিক্ত প্যাকেজ পাবেন না। সাধারণ কোড এডিটিং এর জন্য আপনি এই পদ্ধতি ব্যবহার করতে পারেন। প্রক্রিয়াটি হল:

1. আপনার [নিকটতম GNU মিরর](http://ftpmirror.gnu.org/gnu/emacs/android) বা [মূল GNU FTP সার্ভার](http://ftp.gnu.org/gnu/emacs/android) ভিজিট করুন।

2. আপনার ফোনের CPU আর্কিটেকচার এবং অ্যান্ড্রয়েড ভার্সন অনুযায়ী একটি প্যাকেজ ডাউনলোড করুন।

3. এখন, `.apk` ফাইলটি ইনস্টল করুন। Unknown Source থেকে ইনস্টল করার অনুমতি চাইলে দিয়ে দিন।

**নোট:** [F-Droid-এও](https://f-droid.org/) একটি ভার্সন পাওয়া যায়। তবে এটি অফিসিয়াল ভার্সন থেকে পুরানো। আর, এতে অনেক বাগ ফিক্স ও নতুন ফিচার নেই। তাই অফিসিয়াল APK ব্যবহার করাই ভালো।

### Termux সাপোর্ট সহ ইনস্টল

আপনি Termux সাপোর্ট সহও Emacs ইনস্টল করতে পারেন। Termux হচ্ছে অ্যান্ড্রয়েডে টার্মিনাল ও লিনাক্স চালানোর একটা অ্যাপ। তাই এর সাথে Emacs-কে সমন্বয় করলে দারুণ একটা ব্যাপার হয়। আপনি তখন Termux প্যাকেজ যেমন `git`, `clang`, `python`, `fish` ইত্যাদি Emacs এর সাথে চালাতে পারবেন। আপনার মোবাইলে কোডিং অভিজ্ঞতা হবে আরও উন্নত। তাই আমি পরামর্শ দিব এটা করার, যদিও এতে কিছুটা পরিশ্রম দরকার। তার জন্য আপনার Emacs-এর কী দিয়ে সাইন করা Termux প্যাকেজ দরকার। Emacs সার্ভারেই এমন প্যাকেজ পাওয়া যায়। ইন্সটল করার প্রক্রিয়াটি নিম্নরূপ:

1. আপনার ফোনে আগের Termux এবং Emacs অ্যাপ থাকলে আনইনস্টল করুন।

2. প্রথমে [এই সোর্স](https://sourceforge.net/projects/android-ports-for-gnu-emacs/files/termux/termux-app_apt-android-7-release_universal.apk/download) থেকে Termux ইনস্টল করুন।

3. তারপর [আপনার নিকটতম মিরর](http://ftpmirror.gnu.org/gnu/emacs/android) বা [GNU FTP সার্ভার](http://ftp.gnu.org/gnu/emacs/android) থেকে Emacs ইনস্টল করুন। আপনাকে অবশ্যই `termux/` ডিরেক্টরি থেকে প্যাকেজ ইনস্টল করতে হবে।

4. এখন, Termux এ নিচের কমান্ড রান করুন:

```shell
$ pkg update && pkg upgrade
```
এখানে প্রতিটি প্রশ্নের যথাযথ উত্তর দিন।

5. সব প্যাকেজ ইনস্টল এবং আপডেট হওয়ার পর, Emacs-এ গিয়ে নিচের কোডটি `.emacs.d/early-init.el` ফাইলে যোগ করুন (Early Init File সম্পর্কে আরো জানতে [Emacs ম্যানুয়াল](https://www.gnu.org/software/emacs/manual/html_node/emacs/Early-Init-File.html) দেখুন):

```elisp
(setenv "PATH" (format "%s:%s" "/data/data/com.termux/files/usr/bin"
                        (getenv "PATH")))
(push "/data/data/com.termux/files/usr/bin" exec-path)
```
এটি Emacs এর ভেতরে Termux বাইনারিগুলোর লোকেশন ‍`$PATH` এনভায়রনমেন্ট চলকে যোগ করবে।

এবার, Emacs রিস্টার্ট করুন। এখন আপনি Emacs থেকেই Termux এর প্রোগ্রাম ব্যবহার করতে পারবেন।

**নোট:** Emacs ও Termux (বা এর প্লাগ-ইন) এর যেকোনো ভবিষ্যত আপডেট একই কী দিয়ে সাইন করা থাকতে হবে। Emacs সার্ভার থেকে ডাউনলোড করলে আলাদা করে সাইন করার প্রয়োজন নেই। তবে অন্য কোথাও থেকে প্যাকেজ ডাউনলোড করলে আপনাকে অবশ্যই নিচের লিংকের কী দিয়ে সাইন করে নিতে হবে। এটাই Emacs সার্ভারে ব্যবহার করা হয়।

### Termux রিপোজিটরি থেকে ইনস্টল

আপনি সরাসরি Termux রিপোজিটরি থেকেও Emacs ইনস্টল করতে পারেন। তারপর Termux এর ভিতর থেকেই Emacs ব্যবহার করতে পারবেন। তবে, আমার মতে অফিসিয়াল অ্যাপই উত্তম। কারণ, এটি অ্যান্ড্রয়েড সিস্টেমের মধ্যেই একটি স্বতন্ত্র গ্রাফিক্যাল অ্যাপ হিসেবে চলে। আর এটাতে অ্যান্ড্রয়েডের জন্য ভালো সাপোর্ট আছে। অন্যদিকে, Termux এর Emacs কোন আলাদা অ্যান্ড্রয়েড অ্যাপ নয়। বরং এটা Termux এরই একটি প্যাকেজ। আমি এই প্যাকেজ ব্যবহার করি না এবং এখানে এই পদ্ধতি আলোচনা করব না। আপনি অনলাইনে সার্চ করে টিউটোরিয়াল খুঁজে পেতে পারেন।

## প্রাথমিক জ্ঞান ও সেটআপ

### অ্যান্ড্রয়েডে Emacs কাস্টমাইজ করা

অ্যান্ড্রয়েডে Emacs এর সেটিংস পরিবর্তন করা তথা কাস্টমাইজ করার সেরা উপায় হলো [Easy Customization Interface](https://www.gnu.org/software/emacs/manual/html_node/emacs/Easy-Customization.html) ব্যবহার করা। মেনু বারে গিয়ে **Options > Customize Emacs** থেকে এটি ব্যবহার করা যায়। **Customize Emacs** মেনুতে অনেক অপশন আছে। আপনি **Top-level Emacs Customization Group** দিয়ে শুরু করতে পারেন, যা মূলত **Easy Customization Interface** এর প্রথম পাতা। সেটিংস সার্চ করার জন্য এর উপরে একটি সার্চ বারও আছে। তাছাড়া, আপনি **All Settings Matching** অপশন ব্যবহার করেও সার্চ করতে পারেন।

Emacs কে [Initialization File](https://www.gnu.org/software/emacs/manual/html_node/emacs/Init-File.html) বা সংক্ষেপে Init File দিয়েও কাস্টমাইজ করা যায়। `~/.emacs.el`, `~/.emacs`, বা `~/.emacs.d/init.el` - এই নামগুলো ব্যবহার করে Emacs তার Initialization File খোঁজে। আপনি এই ফাইলে Emacs Lisp নামক প্রোগ্রামিং ভাষা ব্যবহার করে Emacs সেটিংস নিয়ন্ত্রণ করতে পারেন। উদাহরণস্বরূপ, টাইপিং বন্ধ করার ৫ সেকেন্ড পর অটো-সেভ করতে `auto-save-timeout` চলকের মান `setq` দিয়ে পরিবর্তন করুন:

```elisp
(setq auto-save-timeout 5)
```

মডিফায়ার বার চালু করতে `modifier-bar-mode` নামক মাইনর মোড কমান্ডকে আর্গুমেন্ট `1` দিয়ে কল করুন:

```elisp
(modifier-bar-mode 1)
```

Init File এ লেখার নিয়ম সম্পর্কে বিস্তারিত জানতে [Emacs ম্যানুয়াল](https://www.gnu.org/software/emacs/manual/html_node/emacs/Init-Syntax.html) দেখুন। তাছাড়া, Init File লোড হবার পূর্বেই কোন কমান্ড দেবার প্রয়োজন হলে Early Init File ব্যবহার করতে পারেন, যা আমরা [পূর্বে ইন্সটল করার সময় করেছি](#termux-সাপোর্ট-সহ-ইনস্টল)।


পাশাপাশি, Emacs-এর ভেতর থেকেই আপনি ‍`M-x` (`META-x`) চেপে যেকোন কমান্ড রান করতে পারেন। যেমন `M-x Customize` লিখে এন্টার চাপলে **Top-level Emacs Customization Group** চলে আসবে, কারণ `Customize` হচ্ছে **Easy Customization Interface** এর কমান্ড। তাছাড়া, আপনি ট্যাব কমপ্লিশনের মাধ্যমে কমান্ড খুঁজে বের করতে পারবেন।

### Emacs কে স্টোরেজ ব্যবহারের অনুমতি দিন

Emacs এমনিতেই তার অ্যাপ ডাটা ডিরেক্টরি (সাধারণত `/data/data/org.gnu.emacs/files`) ব্যবহার করতে পারে যা তার হোম ডিরেক্টরি হিসেবে ব্যবহৃত হয়। তবে ফাইল খোলা এবং এডিট করার জন্য আপনার অভ্যন্তরীণ স্টোরেজ (`/sdcard`) ব্যবহারের অনুমতি প্রয়োজন। অ্যান্ড্রয়েড ১১+ এ অনুমতি দিতে:

1. **Settings** থেকে **Apps** সেকশনে যান।

2. তারপর, **Special app access** অপশনে যান।

3. অ্যাপের তালিকা থেকে Emacs খুঁজে বের করে তার জন্য **All files access** চালু করুন।

অ্যান্ড্রয়েড ১১ এর নিচের ভার্সনের জন্য, আপনি **App Info** থেকেই স্টোরেজ ব্যবহারের অনুমতি দিতে পারবেন।

তাছাড়া, আপনি Emacs কে অন্যান্য অ্যাপের স্টোরেজ (যেমন ক্লাউড অ্যাপের স্টোরেজ) ব্যবহারের অনুমতিও দিতে পারেন। তার জন্য Emacs মেনু বার থেকে **File > Execute command** এ গিয়ে `android-request-directory-access` কমান্ড দিন। এবার, সংশ্লিষ্ট স্টোরেজ এবং তারপর আপনার পছন্দের ডিরেক্টরি বেছে নিন।

### সর্বদা অন-স্ক্রীন কিবোর্ড চালু রাখুন

অ্যান্ড্রয়েডে Emacs স্ক্রীনের জায়গা বাঁচানোর জন্য রিড-অনলি বাফারে অন-স্ক্রীন কিবোর্ড লুকিয়ে রাখে। আপনি কিবোর্ড দিয়ে যা করতে চান তা সাধারণত মেনু বার বা টুল বার থেকেই করে নিতে পারবেন । তবুও, কিবোর্ড সর্বদা হাতের কাছে থাকা উপকারী। তার জন্য `Customize` এ গিয়ে `touch-screen-display-keyboard` চালু করুন।

### কিবাইন্ডিং নির্ভর কিছু প্যাকেজের জন্য

কিছু Emacs প্যাকেজ (যেমন [evil](https://melpa.org/#/evil)) যেগুলো কিবাইন্ডিংয়ের উপর নির্ভর করে (নির্দিষ্ট বাটনের ভিত্তিতে ক্রিয়া করে), সেগুলো অ্যান্ড্রয়েডের অন-স্ক্রীন কিবোর্ডের সাথে কাজ করে না। কারণ অ্যান্ড্রয়েড ইনপুট মেথড Emacs এর টেক্সট কনভার্সন স্টাইলের উপর নির্ভর করে, যা Emacs এর কিছু কিছু ফিচারের জন্য প্রয়োজন। আপনি যদি এমন কোনো প্যাকেজ ব্যবহার করেন যা এটার কারণে কাজ করে না, তাহলে `overriding-text-conversion-style` চলক দিয়ে গ্লোবালি টেক্সট কনভার্সন বন্ধ করুন অথবা `text-conversion-style` দিয়ে লোকালি নির্দিষ্ট বাফারে বন্ধ করুন। এ ব্যাপারে আরো জানতে [Elisp ম্যানুয়ালের](https://www.gnu.org/software/emacs/manual/html_node/elisp/index.html#SEC_Contents) [Misc Events](https://www.gnu.org/software/emacs/manual/html_node/elisp/Misc-Events.html) দ্রষ্টব্য।

### দৃশ্যমান বারগুলো সেট আপ করুন

আপনি যদি ফিজিক্যাল কিবোর্ড ছাড়া অ্যান্ড্রয়েডে Emacs ব্যবহার করতে চান, বিভিন্ন দৃশ্যমান বার যেমন মেনু বার, টুল বার ইত্যাদি হবে আপনার সেরা বন্ধু। তাই `Customize` থেকে মেনু বার, টুল বারসহ আরও যা যা প্রয়োজন চালু করে দিন। অ্যান্ড্রয়েডের জন্য একটি খুবই উপকারী অপশন হলো Emacs এর মডিফায়ার বার। এতে মডিফায়ার কী (যেমন CTRL, ALT, META ইত্যাদি) এর বাটন থাকে। আপনি `tool-bar-mode` চালু থাকা অবস্থায় `modifier-bar-mode` দিয়ে এটি চালু করতে পারেন। কারণ, মডিফায়ার বারটি টুল বারের সাথে যুক্ত। যখনই কোনো মডিফায়ার কী টাচ করবেন, Emacs অন-স্ক্রীন কিবোর্ডও দেখাবে এবং টেক্সট কনভার্সন সাময়িকভাবে বন্ধ করে রাখবে, যাতে আপনি Text বা Prog Mode বাফারে মডিফায়ার ব্যবহার করতে পারেন। আপনি `tool-bar-position` এর মান `bottom` সেট করে এই কী-গুলো আপনার আঙুলের কাছাকাছি রাখতে পারেন।

### Emacs দিয়ে ফাইল ওপেন করা

আপনি যখন অ্যান্ড্রয়েডে কোন ফাইল ওপেন করার চেষ্টা করেন, এটি জিজ্ঞাসা করে কোন অ্যাপ ব্যবহার করতে চান। Emacs দিয়ে কোন ফাইল ওপেন করতে আপনার Emacs সার্ভার চালু করতে হবে। Emacs এর ভিতর থেকে `M-x server start` রান করে সার্ভার চালু করতে হয়। তবে Emacs খুললেই সার্ভার অটোমেটিকালি চালু করা আপনার জন্য সুবিধাজনক হবে। এটি করতে, আপনার [Initialization File](https://www.gnu.org/software/emacs/manual/html_node/emacs/Init-File.html) এ `(server-start)` যোগ করুন।

### পড়া ও এডিটিং উন্নত করা

কোন ফাইলে লম্বা লাইন থাকলে তা সহজে পড়ার উপায় হচ্ছে ওয়ার্ড-র‍্যাপ ব্যবহার করা। তাহলে লাইনগুলো স্ক্রীন অতিক্রম করার বদলে কয়েক লাইনে দেখাবে যাতে ডানে-বামে স্ক্রল করতে না হয়। `visual-line-mode` দিয়ে আপনি এরূপ লাইনগুলোর প্রকৃত ক্রমের বদলে দৃশ্যমান ক্রমানুসারে এডিট করতে পারবেন। এই মোড সাধারণ এডিটিং কমান্ডগুলোকে দৃশ্যমান লাইনে কাজ করতে দেয় এবং বাফারে ওয়ার্ড-র‍্যাপও চালু করে। ফলে আপনি অ্যান্ড্রয়েডে ভালোভাবে পড়তে এবং টাইপ করতে পারবেন। পাশাপাশি, আপনি `visual-wrap-prefix-mode` চালু করে পরবর্তী র‍্যাপ করা লাইনগুলো ইন্ডেন্টও করতে পারেন। পাশাপাশি, আপনি নিচের ফিচারগুলোও দেখতে পারেন:

- [Visual fill](https://elpa.gnu.org/packages/visual-fill.html)

- [Adaptive filling](https://www.gnu.org/software/emacs/manual/html_node/emacs/Adaptive-Fill.html)

- [Adaptive wrap](https://elpa.gnu.org/packages/adaptive-wrap.html)

### ফন্ট সেট করা

আপনি `set-frame-font` ফাংশন দিয়ে Emacs এ ডিফল্ট ফন্ট সেট করতে পারেন:

```elisp
(set-frame-font "JetBrains Mono")
```

পাশাপাশি, আপনি `Customize` এর **Faces** গ্রুপ থেকেও ফন্ট সেট করতে পারবেন। Emacs এর বিভিন্ন অংশে ব্যবহৃত ফন্টের জন্য এই গ্রুপের অধীনে বিভিন্ন সাবগ্রুপ আছে। আপনি `Basic Faces` সাবগ্রুপ দিয়ে শুরু করতে পারেন এবং প্রয়োজন অনুযায়ী অন্যগুলো কাস্টমাইজ করতে পারেন। আপনার মূল ফন্ট বাংলা সাপোর্ট না করলে বাংলা ভাষার জন্য ফন্টসেট নির্ধারণ করতে হবে। তার জন্য আপনার Init File এ নিচের কোডটি ব্যবহার করতে পারেন:

```elisp
     (set-fontset-font "fontset-default" nil "Noto Sans Bengali" nil 'append)
```

এই কোড অনুসারে Emacs বাংলা ভাষার জন্য Noto Sans Bengali ফন্টটি ব্যবহার করবে। এই ফন্টটি সাধারণত অ্যান্ড্রয়েড ফোনে আগে থেকেই দিয়ে দেওয়া থাকে। তবে আপনি যে ফন্টটি ব্যবহার করতে চান তা যদি ডিভাইসে না থাকে, তাহলে Emacs এর হোম ডিরেক্টরিতে `fonts` নামে একটি ফোল্ডার তৈরি করে সেখানে ফন্ট ফাইলটি কপি করতে হবে।

### কোড লেখা ছাড়াও আর যা করা যায়

ইতোপূর্বে বলেছিলাম, Emacs এ আপনি কেবল কোডিং ছাড়াও আরো অনেক কিছুই করতে পারবেন। তার জন্য Emacs এর সাথেই কিছু প্যাকেজ দেয়া থাকে। পাশাপাশি, আপনি অন্যান্য প্যাকেজ ইন্সটল করে আরও ফিচার যুক্ত করতে পারেন। Emacs এ কি কি করা যায় তার সংক্ষেপে কিছু উদাহরণ আমি এখানে পেশ করছি। বিস্তারিত জানতে সংশ্লিষ্ট ম্যানুয়াল পড়তে পারেন।

- কর্মপরিকল্পনা: আপনি Org Mode ব্যবহার করে Emacs এ নিজের কাজকর্মের পরিকল্পনা তৈরি করতে পারবেন। Org Mode হচ্ছে Emacs এর একটি লেখনী টুল ও কর্মতালিকা ম্যানেজার। Emacs সম্পর্কিত আলোচনায় এই ফিচারটি প্রায়ই উঠে আসে। এটি মূলত একটি মার্কআপ ভাষা- বলতে পারেন মার্কডাউনের বিকল্প। তবে এর সিনট্যাক্স মার্কডাউনের তুলনায় সহজ। এর কিছু কাজের ফিচার রয়েছে এবং এটা Emacs এর সাথে ইন্টিগ্রেটেড। আপনি হেডিং লাইনে TAB চেপে তার সাবহেডিংগুলো লুকিয়ে রাখতে পারবেন। বারবার TAB চাপলে সেগুলোকে ধাপ অনুসারে দৃশ্যমান করা হবে। বাফারের যেকোন জায়গায় `S-TAB` চাপলে পুরো ফাইলের জন্য কাজ করবে। এ সম্পর্কে বিস্তারিত জানতে Org Mode ম্যানুয়াল পড়ুন।

- মেইল ও সংবাদ পাঠ: Emacs এ Gnus এর মাধ্যমে ইমেইল ও সংবাদ পড়া যায়। যদিও এই প্রোগ্রামটিতে শুরুতে কেবল ইউজনেটের মত সংবাদই পড়া যেত, তবে এখন এটি একটি ইমেইল ক্লায়েন্ট হিসেবেও কাজ করে। তবে এটিতে মেইল পড়াও নিউজগ্রুপের মত। বিস্তারিত জানতে Gnus ম্যানুয়াল পড়তে পারেন। Gnus ছাড়াও আপনি [mu4e](https://www.djcbsoftware.nl/code/mu/mu4e.html) প্যাকেজ ব্যবহার করে মেইল পড়তে পারবেন।

- ক্যালেন্ডার ও ডায়েরি: Emacs এ একটি ডেস্ক ক্যালেন্ডার ফিচার রয়েছে, যার সাথে ইভেন্ট রেকর্ড করার জন্য ডায়েরিও আছে। এটা দিয়ে অ্যাপয়েন্টমেন্ট ম্যানেজ করা এবং টাইম ট্র্যাকও করা যায়। ক্যালেন্ডার দেখতে `M-x calendar` কমান্ড দিন। ক্যালেন্ডার থেকে বের হতে `q` চাপুন।

- বই পড়া: Emacs এ আপনি বইও পড়তে পারবেন। এটা PDF,
OpenDocument, Microsoft Office, EPUB, PostScript (PS) ইত্যাদি বিভিন্ন ফাইল ফরম্যাট সাপোর্ট করে। আপনি এতে জুম, সার্চ, পৃষ্টা ভাগ করা ইত্যাদি ফিচার পাবেন।

- ফাইল ম্যানেজার: Emacs এর একটি নিজস্ব ফাইল ম্যানেজার আছে, যার নাম Dired। এটি খুবই সরল এবং কোডিংয়ের সময় সহজেই ফাইল খুঁজে বের করতে কাজে লাগে।

### প্যাকেজ ইন্সটল করা

Emacs এ আপনি প্যাকেজ ইন্সটল করে ফিচার যুক্ত করতে পারবেন। এর জন্য `package-install` কমান্ড ব্যবহার করতে হবে। প্যাকেজ সম্পর্কে আরো তথ্যের জন্য `list-packages` কমান্ড ব্যবহার করুন। এখানে কয়েকটি কমান্ড রয়েছে। যেমন `i` দিয়ে ইন্সটল আর `d` দিয়ে ডিলিট করার জন্য চিহ্নিত করা, `x` দিয়ে চিহ্নিত প্যাকেজ ইন্সটল করা, `?`/`RET` চেপে প্যাকেজের বিবরণ দেখা ইত্যাদি। Emacs দুটি জায়গা থেকে প্যাকেজ ডাউনলোড করে: [GNU ELPA](https://elpa.gnu.org/) আর [NonGNU ELPA](https://elpa.nongnu.org/)। তবে আপনি বাড়তি আর্কাইভ (যেমন [MELPA](https://melpa.org)) যোগ করতে পারবেন। তার জন্য `package-archives` চলকটির মান পরিবর্তন করতে হবে।

### Emacs ম্যানুয়াল পড়ুন

Emacs এ অনেক ফিচার রয়েছে এবং আপনার মাঝে মাঝেই সাহায্যের প্রয়োজন হবে। আর সাহায্য পাওয়ার সেরা উপায় হলো ম্যানুয়াল পড়ে *নিজের সাহায্য নিজেই করা*। Emacs এর সাথেই বেশ কিছু ম্যানুয়াল অন্তর্ভুক্ত আছে:

- [The Emacs Manual](https://www.gnu.org/software/emacs/manual/html_node/emacs/index.html)

- [An Introduction to Programming in Emacs Lisp](https://www.gnu.org/software/emacs/manual/html_node/eintr/index.html)

- [Emacs Lisp Reference Manual](https://www.gnu.org/software/emacs/manual/html_node/elisp/index.html)

- এবং [অন্যান্য ম্যানুয়াল](https://www.gnu.org/software/emacs/manual/index.html)

আপনি `C-h i` কমান্ড দিয়ে Emacs এর ভিতর থেকে বা [অনলাইনে](https://www.gnu.org/software/emacs/manual/index.html) এগুলো পড়তে পারেন। `C-h r` কমান্ড সরাসরি Emacs ম্যানুয়াল ওপেন করে। ম্যানুয়ালের ভিতর `i` টাইপ করে আপনি হেল্প টপিক খুঁজে বের করতে পারেন যা খুবই সুবিধাজনক। এ জায়গা থেকে সহজেই অ্যান্ড্রয়েড বিষয়ক টপিকগুলে ম্যানুয়াল থেকে খুঁজে পেতে পারেন। অ্যান্ড্রয়েডের জন্য ম্যানুয়ালের একটি আলাদা অংশই রয়েছে - [Appendix H Emacs and Android](https://www.gnu.org/software/emacs/manual/html_node/emacs/Android.html)। আপনি বিশেষ করে নিচের অংশগুলো পড়তে পারেন:

- [Accessing Files from Other Programs on Android](https://www.gnu.org/software/emacs/manual/html_node/emacs/Android-Document-Providers.html)

- [What Files Emacs Can Access on Android](https://www.gnu.org/software/emacs/manual/html_node/emacs/Android-File-System.html)

- [Running Emacs under Android](https://www.gnu.org/software/emacs/manual/html_node/emacs/Android-Environment.html)

- [Installing Extra Software on Android](https://www.gnu.org/software/emacs/manual/html_node/emacs/Android-Software.html)

- [Touchscreen Input and Virtual Keyboards](https://www.gnu.org/software/emacs/manual/html_node/emacs/Other-Input.html)

- [Touchscreen Events](https://www.gnu.org/software/emacs/manual/html_node/elisp/Touchscreen-Events.html)

তাছাড়া আপনি `describe-*` কমান্ড যেমন `describe-function`, `describe-mode`, `describe-variable` ইত্যাদি দিয়ে বিভিন্ন ফাংশন, চলক, মোড ইত্যাদির বিবরণ এবং তাদের মান সম্পর্কে জানতে পারেন। শুরু করতে শুধু `M-x` টাইপ করুন বা **File > Execute Command** অপশন ব্যবহার করুন।
